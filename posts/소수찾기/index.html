[Progrmmers] Level 2: 소수 찾기

Lee Hyun Kyeong on Dec 20, 20212021-12-20T12:33:00+09:00
Updated Jan 112022-01-11T17:30:27+09:00 6 min read

📖 문제 설명

한자리 숫자가 적힌 종이 조각이 흩어져있습니다. 흩어진 종이 조각을 붙여 소수를 몇 개 만들 수 있는지 알아내려 합니다.

각 종이 조각에 적힌 숫자가 적힌 문자열 numbers가 주어졌을 때, 종이 조각으로 만들 수 있는 소수가 몇 개인지 return 하도록 solution 함수를 완성해주세요.

💥 제한사항

  • numbers는 길이 1 이상 7 이하인 문자열입니다.
  • numbers는 0~9까지 숫자만으로 이루어져 있습니다.
  • “013”은 0, 1, 3 숫자가 적힌 종이 조각이 흩어져있다는 의미입니다.

🤯 풀이 로직

  1. 에라토스테네스의 체로 0~9999999까지 소수를 구해 놓는다. (boolean으로 true: 소수, false: 소수가 아님)
  2. 1개~numbers 길이만큼 순열을 뽑는다.

문제에서 011은 11과 같다고 하였지만, 이를 따로 처리하지 않아도 된다. 그 이유는 나는 isPrime 배열의 인덱스를 통해 소수를 구해놓았다. 즉, 인덱스가 곧 수이다. 011이라는 인덱스는 없기때문에 011은 소수인지 판단하지 않고 11만 소수인지 판단해주면된다. 또한, [1, 1]과 같은 입력이 주어졌을 때 11(앞의 1, 뒤의 1), 11(뒤의 1, 앞의 1) 과 같이 같은 수를 중복해서 뽑는 경우를 처리해주기 위해서 한 번 소수라고 판단하면 그 수는 소수에서 제외(false로 처리)하여 중복을 방지하고자 하였다.

필요한 함수

  1. 에라토스테네스의 체를 구현하는 함수
  2. 순열 함수

💭 회고

이 문제 역시 java로 언어를 바꾸면서 로직을 풀어내기까지 1시간이 걸렸다..! 배열 초기화하는 법부터 문자열 자르고 String을 String 배열로 만들고 등등.. 최대한 ide를 사용하지 않으려고 했으나,, 디버깅을 안 할수가 없어서 intellij로 한 줄 씩 디버깅 찍으면서 오류를 차차 해결해나갔다. tmp.substring(0, tmp.length()-1)을 이용해서 문자열을 자를때 나는 tmp 자체가 잘릴 줄 알았는데, 자르고 다시 tmp에 넣어줘야지만 결과가 반영이 되었다. java도 다른 언어처럼 String형이어도 배열처럼 접근 가능하다고 생각했는데 따로 처리를 해주어야 배열처럼 접근할 수 있었다. 한 문제에서 java의 많은 알게된 것 같은기분이다..!

🌠 배운점

  • Arrays.fill(배열, 값) 을 통해 배열을 초기화하는 법을 배웠다.
  • 에라토스테네스의 체를 java로 구현하는 법을 배웠다.
  • 순열 구하는 함수를 java로 구현하는 법을 배웠다.
  • Integer.parseInt(string)을 통해 String형을 int형으로 바꾸는 법을 배웠다.
  • str.substring(start index, end index)를 통해 문자열을 자르는 법을 배웠다.
  • str.split("")을 통해 String을 String 배열로 바꾸는 법을 배웠다.
  • length는 배열에!! length()는 문자열에!! 사용하는 법을 배웠다.

import java.util.*;

class Solution {
    boolean[] isPrime = new boolean[10000000]; 
    int answer = 0;
    public int solution(String numbers) {
        boolean[] visited = new boolean[numbers.length()]; // 방문했는지 확인
        Arrays.fill(isPrime, true); // default: 소수
        isPrime[0] = isPrime[1] = false; // 0과 1은 소수가 아님
        eratos(); // 소수 구해 놓는다.
        for(int i=1; i<=numbers.length(); i++) {
            // 1개~numbers의 길이만큼 뽑는다.
            Arrays.fill(visited, false); // 방문 초기화
            permutation(numbers.split(""), visited, "", 0, i); // (문자열 -> 배열, 방문 확인, 0개부터 시작, 뽑을 개수)
        }
        return answer;
    }
    
    public void eratos() {
          for (int i=2;i<10000000;i++) {
              if (isPrime[i]) { // 소수면 
                  // 해당 수의 배수들은 모두 소수가 아니다.
                  for (int j=2; j*i < 10000000; j++) {
                      isPrime[i*j] = false;
                  }
              }
          }
    }
    
    public void permutation(String[] numbers, boolean[] visited, String tmp, int idx, int cnt) {
        if (tmp.length() == cnt) {
            if (isPrime[Integer.parseInt(tmp)]) {
                answer++;
                isPrime[Integer.parseInt(tmp)] = false; // 중복 방지
            }
            return;
        }
        for (int i=0; i < numbers.length; i++) {
            if (visited[i]) continue;
            visited[i] = true;
            tmp += numbers[i];
            permutation(numbers, visited, tmp, i, cnt); // 재귀
            tmp = tmp.substring(0, tmp.length()-1);
            visited[i] = false;
        }
    }
}
