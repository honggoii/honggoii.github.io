[백준] 14567번: 선수과목 (Prerequisite)

Lee Hyun Kyeong on Jan 92022-01-09T04:03:06+09:00
Updated Jan 122022-01-12T22:46:49+09:00 7 min read

📖 문제 설명

올해 Z대학 컴퓨터공학부에 새로 입학한 민욱이는 학부에 개설된 모든 전공과목을 듣고 졸업하려는 원대한 목표를 세웠다. 어떤 과목들은 선수과목이 있어 해당되는 모든 과목을 먼저 이수해야만 해당 과목을 이수할 수 있게 되어 있다. 공학인증을 포기할 수 없는 불쌍한 민욱이는 선수과목 조건을 반드시 지켜야만 한다. 민욱이는 선수과목 조건을 지킬 경우 각각의 전공과목을 언제 이수할 수 있는지 궁금해졌다. 계산을 편리하게 하기 위해 아래와 같이 조건을 간소화하여 계산하기로 하였다.

  1. 한 학기에 들을 수 있는 과목 수에는 제한이 없다.
  2. 모든 과목은 매 학기 항상 개설된다.

모든 과목에 대해 각 과목을 이수하려면 최소 몇 학기가 걸리는지 계산하는 프로그램을 작성하여라.

💦 입력

첫 번째 줄에 과목의 수 N(1≤N≤1000)과 선수 조건의 수 M(0≤M≤500000)이 주어진다. 선수과목 조건은 M개의 줄에 걸쳐 한 줄에 정수 A B 형태로 주어진다. A번 과목이 B번 과목의 선수과목이다. A<B인 입력만 주어진다. (1≤A<B≤N)

🍩 출력

1번 과목부터 N번 과목까지 차례대로 최소 몇 학기에 이수할 수 있는지를 한 줄에 공백으로 구분하여 출력한다.

🤯 풀이 로직

  1. 입력을 받는다.
  2. indegree가 0인 애들을 큐에 넣는다.
  3. 정답을 구한다.
  4. 정답을 출력한다.

필요한 자료형

  • 과목 수 : int n
  • 조건 수 : int m
  • 학기수를 저장할 배열 : int[] answer = new int[n+1]
  • 들어오는 간선 개수 :int indegree = new in[n+1]
  • 큐 : Queue q = new LinkedList()
  • 그래프 : LinkedList<Integer>[] graph = new ArrayList[n+1]

필요한 함수

  1. 입력 함수
    • a와 b를 입력
    • b과목의 indegree에 1 더하기
    • a에 b를 연결
  2. indegree가 0인 애들을 큐에 넣고 시작하는 함수
    • 1~n까지 과목을 확인
      • 현재 과목의 indegree가 0이면
    • 큐에 넣는다.
    • 현재 과목의 학기수를 1로 초기화
  3. 정답을 구하는 함수
    • 큐가 빌때까지 반복
      • 큐에서 하나 꺼낸다.
      • 현재 과목에 연결된 과목을 확인한다.
    • 연결된 과목의 indegree를 1 감소
    • 만약 연결된 과목의 indegree가 0이 되었으면
      • 큐에 넣는다.
      • 연결된 과목의 학기수는 현재 과목의 학기수에 1을 더하여 저장

💭 회고

내가 좋아하는(쉽게만 나온다면 ㅎㅎ;) 위상정렬 문제를 풀어보았다❗ 위상정렬을 구현하는 기본 문제라서 어렵지는 않았는데 java에서 Queue를 선언하는 부분은 검색을 통해서 알게되었다. 문제를 풀고 빌드하니까 널 포인터 에러가 났는데 2차원 배열 생성이 제대로 되지 않아서였다..! 열만 만들고 각 열의 행을 만들어주지 않아서 난 오류였다.

import java.io.*;
import java.util.*;

public class Main {
    static int n; // 과목 수
    static int m; // 조건 수
    static int[] answer; // 정답 출력 배열
    static int[] indegree; // 들어오는 간선 개수
    static ArrayList<Integer>[] graph; // 그래프
    static Queue q = new LinkedList();

    static void input() throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String[] tokens = br.readLine().split(" ");
        n = Integer.parseInt(tokens[0]);
        m = Integer.parseInt(tokens[1]);
        answer = new int[n+1];
        indegree = new int[n+1];
        graph = new ArrayList[n+1];

        for (int i=0; i<n+1; i++) {
            graph[i] = new ArrayList<>(); // 2차원 배열 생성
        }

        for (int i=0; i<m; i++) {
            tokens = br.readLine().split(" ");
            int a = Integer.parseInt(tokens[0]);
            int b = Integer.parseInt(tokens[1]);
            indegree[b]++;
            graph[a].add(b);
        }
    }

    static void init() {
        for (int i=1; i<=n; i++) {
            if (indegree[i] == 0) {
                q.add(i);
                answer[i] = 1;
            }
        }
    }

    static void start() {
        while (!q.isEmpty()) {
            int now = (int) q.poll();

            for (int next: graph[now]) {
                indegree[next]--;

                if (indegree[next] == 0) {
                    q.add(next);
                    answer[next] = answer[now] + 1;
                }
            }
        }
    }

    static void output() {
        for (int i=1; i<=n; i++) {
            System.out.print(answer[i]);
            System.out.print(" ");
        }
    }

    public static void main(String[] args) throws IOException {
        input(); // 입력
        init(); // indegree가 0인 애들을 큐에 넣고 시작
        start(); // 정답 구하러
        output(); // 출력
    }
}
