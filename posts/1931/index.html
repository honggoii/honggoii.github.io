[백준] 1931번: 회의실 배정

Lee Hyun Kyeong on Jan 122022-01-12T22:47:06+09:00
Updated Jan 132022-01-13T00:32:39+09:00 7 min read

📖 문제 설명

한 개의 회의실이 있는데 이를 사용하고자 하는 N개의 회의에 대하여 회의실 사용표를 만들려고 한다. 각 회의 I에 대해 시작시간과 끝나는 시간이 주어져 있고, 각 회의가 겹치지 않게 하면서 회의실을 사용할 수 있는 회의의 최대 개수를 찾아보자. 단, 회의는 한번 시작하면 중간에 중단될 수 없으며 한 회의가 끝나는 것과 동시에 다음 회의가 시작될 수 있다. 회의의 시작시간과 끝나는 시간이 같을 수도 있다. 이 경우에는 시작하자마자 끝나는 것으로 생각하면 된다.

💦 입력

첫째 줄에 회의의 수 N(1 ≤ N ≤ 100,000)이 주어진다. 둘째 줄부터 N+1 줄까지 각 회의의 정보가 주어지는데 이것은 공백을 사이에 두고 회의의 시작시간과 끝나는 시간이 주어진다. 시작 시간과 끝나는 시간은 231-1보다 작거나 같은 자연수 또는 0이다.

🍩 출력

첫째 줄에 최대 사용할 수 있는 회의의 최대 개수를 출력한다.

🤯 풀이 로직

  1. 회의의 시작 시간과 끝나는 시간을 가지는 클래스를 하나 만든다.
  2. 입력을 받는다.
  3. 클래스 배열을 정렬한다.
  4. 정답을 구한다.
  5. 정답을 출력한다.

필요한 자료형

  • 회의 수 : int n
  • 회의 클래스 : Meeting(int start, int end)
  • 사용할 수 있는 회의 최대 수(정답) : int answer
  • 회의 정보를 담은 배열: Meeting[] info = new Meeting[n]
  • 다음 회의 시작 시간 : int next

필요한 클래스 및 함수

  1. 회의 클래스
    • Comparable을 구현해야한다.
    • 멤버 변수 : int start, int end
    • compareTo() 메소드를 오버라이드 한다.
      • end를 기준으로 오름차순 정렬하고 만약 end가 같으면 start를 기준으로 오름차순 정렬한다.
  2. 정답 구하는 함수
    • 다음 회의 시작 시간을 회의 정보 배열 첫번째의 끝나는 시간으로 둔다.
    • 사용할 수 있는 회의 수 1증가
    • 인덱스 1~(n-1)까지
      • 회의 정보 배열의 혀재 인덱스의 시작 시간이 다음 회의 시작 시간 이후면(같거나 크면)
    • 다음 회의 시작 시간을 회의 정보 배열의 현재 인덱스의 끝나는 시간으로 세팅한다.
    • 사용할 수 있는 회의 수 1증가

사용할 수 있는 회의의 수를 최대로 하기 위해서는 끝나는 시간이 빠른 회의를 여러개하면 된다. 그래서 끝나는 시간 기준으로 오름차순 정렬을 먼저 해주고, 만약 끝나는 시간이 같으면 시작 시간이 빠를 수록 더 많은 회의를 할 수 있으므로 시작 시간을 기준으로 오름차순 정렬을 해준다.

💭 회고

어제 코테 1번 문제에서 멘붕와서 좌절했는데 비슷한 문제를 알려주셔서 오늘 풀어보았다. 한정된 시간안에 회의를 많이 하기 위해서는 끝나는 시간이 빠를수록, 끝나는 시간이 같으면 시작 시간이 빠를수록 유리하다는 것을 알고 그리디하게 푸는 방법을 배울 수 있는 문제였다! 어제 테케가 계속 터졌던 것을 반성하며.. 이 문제는 까먹지 않길….!

import java.io.*;
import java.util.*;

public class Main {
    static int n; // 회의 수
    static int answer; // 사용할 수 있는 회의 최대 수
    static Meeting[] info; // 회의 정보를 담은 배열
    static int next; // 다음 회의 시작 시간

    // 회의 클래스
    static class Meeting implements Comparable<Meeting> {
        int start; // 회의 시작 시간
        int end; // 회의 끝나는 시간

        public Meeting(int start, int end) {
            this.start = start;
            this.end = end;
        }

        @Override
        public int compareTo(Meeting o) {
            if (end == o.end) return start - o.start; // 시작 시간 오름차순
            return end - o.end; // 끝나는 시간 오름차순
        }
    }

    static void input() throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String[] tokens = br.readLine().split(" ");
        n = Integer.parseInt(tokens[0]);
        info = new Meeting[n];
        for (int i=0; i<n; i++) {
            tokens = br.readLine().split(" ");
            info[i] = new Meeting(Integer.parseInt(tokens[0]), Integer.parseInt(tokens[1]));
        }
    }

    static void sorting() {
        Arrays.sort(info);
    }

    static void solve() {
        next = info[0].end; // 다음 회의 시작 시간
        answer++;

        for (int i=1; i<n; i++) {
            if (next <= info[i].start) {
                next = info[i].end; // 다음 회의 시작 시간 갱신
                answer++;
            }
        }
    }

    public static void main(String[] args) throws IOException {
        input(); // 입력
        sorting(); // 회의 클래스 정렬
        solve(); // 정답 구하기
        System.out.println(answer);// 정답 출력
    }
}
