[백준] 1260번: DFS와 BFS

Lee Hyun Kyeong on Dec 29, 20212021-12-29T00:35:42+09:00
Updated Jan 122022-01-12T22:46:49+09:00 5 min read

📖 문제 설명

그래프를 DFS로 탐색한 결과와 BFS로 탐색한 결과를 출력하는 프로그램을 작성하시오. 단, 방문할 수 있는 정점이 여러 개인 경우에는 정점 번호가 작은 것을 먼저 방문하고, 더 이상 방문할 수 있는 점이 없는 경우 종료한다. 정점 번호는 1번부터 N번까지이다.

💦 입력

첫째 줄에 정점의 개수 N(1 ≤ N ≤ 1,000), 간선의 개수 M(1 ≤ M ≤ 10,000), 탐색을 시작할 정점의 번호 V가 주어진다. 다음 M개의 줄에는 간선이 연결하는 두 정점의 번호가 주어진다. 어떤 두 정점 사이에 여러 개의 간선이 있을 수 있다. 입력으로 주어지는 간선은 양방향이다.

🍩 출력

첫째 줄에 DFS를 수행한 결과를, 그 다음 줄에는 BFS를 수행한 결과를 출력한다. V부터 방문된 점을 순서대로 출력하면 된다.

🤯 풀이 로직

  1. 간선 연결
  2. dfs
  3. 방문 초기화
  4. bfs

필요한 자료형

  • 정점 개수 : int n
  • 간선 개수 : int m
  • 탐색 시작 정점 : int v
  • 그래프 : ArrayList<Integer>[] graph = new ArrayList[n+1] => graph[i] = new ArrayList<>()
  • 방문 확인 : boolean[] visited = new boolean[n+1]

필요한 함수

  1. 간선 연결 함수
    • 두 정점 입력
      • 각 정점에 서로의 간선 추가 (양방향 연결)
  2. dfs
    • 매개변수 받은 정점 방문 체크
    • 정답에 정점 추가
    • 정점에 연결된 애들 하나씩 순회
      • 방문한 적 없으면
    • dfs 재귀
  3. 방문 초기화 함수
    • 인덱스 0 ~ n까지 방문 체크하는 배열을 모두 false로 초기화
  4. bfs
    • 큐 선언
    • 큐에 매개변수로 받은 정점 추가
    • 매개변수로 받은 정점 방문 체크
    • 큐가 빌때까지 반복
      • 큐에서 하나 꺼냄
      • 정답에 추가
      • 큐에서 꺼낸 정점과 연결된 정점 순회
    • 방문한 적 없으면
      • 큐에 추가
      • 방문 체크

💭 회고

bfs와 dfs를 java로 처음 구현해보면서 bfs와 dfs 로직을 다시 떠올리게 되었다. java에서 queue 사용하는 법도 배웠다! java로 알고리즘 풀기 시작한 이후로 입출력부터 문제였는데 그래도 점점 익숙해지는 것 같아서 뿌듯하다 ㅎㅎ 꾸준히 몸에 익히면 알아서 반응할거다..! 좀 익숙해지면 다시 python을 서브로 할 수 있도록 감을 잃지 말아야겠다!🚀

🌠 배운것

  • java로 Queue를 사용하는 방법을 배웠다.
  • java로 dfs, bfs 구현하는 방법을 배웠다.

import java.io.*;
import java.util.*;

public class Main {
    static int n; // 정점
    static int m; // 간선
    static int v; // 탐색 시작 정점
    static ArrayList<Integer>[] graph; // 그래프
    static boolean visited[]; // 방문 확인
    static StringBuilder sb = new StringBuilder(); // 정답

    static void input() throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String[] tokens = br.readLine().split(" ");
        n = Integer.parseInt(tokens[0]);
        m = Integer.parseInt(tokens[1]);
        v = Integer.parseInt(tokens[2]);
        graph = new ArrayList[n+1];
        visited = new boolean[n+1];
        for (int i=1; i<n+1; i++) {
            graph[i] = new ArrayList<>();
        }

        for (int i=0; i<m; i++) {
            tokens = br.readLine().split(" ");
            int a = Integer.parseInt(tokens[0]);
            int b = Integer.parseInt(tokens[1]);

            // 양방향 연결
            graph[a].add(b);
            graph[b].add(a);
        }

        // 작은 숫자부터 방문
        for (int i=1; i<n+1; i++) {
            Collections.sort(graph[i]);
        }
    }

    static void dfs(int vertex) {
        visited[vertex] = true;
        sb.append(vertex).append(" ");

        for (int v: graph[vertex]) {
            if (!visited[v]) {
                dfs(v);
            }
        }
    }

    static void initVisited() {
        for (int i=0; i<n+1; i++) {
            visited[i] = false;
        }
    }

    static void bfs(int vertex) {
        Queue<Integer> q = new LinkedList<>();
        q.add(vertex);
        visited[vertex] = true;

        while(!q.isEmpty()) {
            int now = q.poll();
            sb.append(now).append(" ");

            for (int nxt: graph[now]) {
                if (!visited[nxt]) {
                    q.add(nxt);
                    visited[nxt] = true;
                }
            }
        }
    }

    public static void main(String[] args) throws IOException {
        input();
        dfs(v);
        sb.append("\n");
        initVisited(); // 방문 초기화
        bfs(v);
        System.out.println(sb);
    }

}
