# 최대공약수 x 최소공배수 = 두 수의 곱

## 알고리즘
1. a%b가 0이 될 때까지 재귀 (a와 b의 대소를 비교해줄 필요는 없다.)

```java
int gcd(int a, int b) {
    if (b == 0) return a;
    return gcd(b, a%b);
}
```

만약 a=15, b=12라고 한다면,

gcd(15, 12) => gcd(12, 3) => gcd(3, 0) => return 3;
