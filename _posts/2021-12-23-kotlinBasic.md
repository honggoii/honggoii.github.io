## repeat()
argument에 수를 넣어주면 해당 수 만큼 반복을 할 수 있다.
```kotlin
fun main() {
    repeat(5) {
	    println("Hello, world!")    
    }
}
```
```
// 출력결과
Hello, world!
Hello, world!
Hello, world!
Hello, world!
Hello, world!
```
